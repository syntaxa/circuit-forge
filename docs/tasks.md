# Задачи

## Доработки и развитие
- Добавить описания упражнений на экран тренировки.
- Смена стороны во время тренировки для двухсторонних упражнений.
- "3 2 1 — дзинь!" + придумать идею, как уместить длинные названия упражнений в голосовом объявлении.
- Голосовой анонс "Далее — упражнение" на самом первом цикле (сейчас первое упражнение не объявляется).
- Отмена активной тренировки: пауза, подтверждение отмены, отмена.
- Добавить короткое название упражнения (`shortName`) для TTS и fallback-правило при длинных названиях.
- Визуальная индикация смены стороны (текст/иконка + тайминг).
- Очищать/останавливать очередь TTS при паузе, отмене и выходе с экрана тренировки.
- Добавить настройку длительности подготовительного отсчёта (3‑2‑1).

## Тестирование
- Не забываются группы мышц последней тренировки и корректно используются в составлении плана новой тренировки. Запись мышц прошлой тренировки только после события "полностью выполнена тренировка".
- При отмене тренировки история и группы мышц не обновляются.
- Первое упражнение озвучивается: "Далее — <название>" + 3‑2‑1.
- Смена стороны озвучивается и визуально отображается ровно в нужный момент.
