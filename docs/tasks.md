# Задачи




## Настройка групп мышц для тренировки
- [ ] Возможность менять состав групп мышц для новой тренировки. Сейчас по умолчанию берутся 2 группы из предыдущей тренировки + 1 новая. Добавить настройку: количество «закреплённых» групп из прошлой тренировки — 1, 2 или 3 (остальные выбираются новыми).
